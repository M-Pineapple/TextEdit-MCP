# TextEdit MCP Example

This example demonstrates all the formatting features available in TextEdit MCP.

## Text Formatting

- **Bold text** for emphasis
- *Italic text* for style
- ***Bold and italic*** combined
- ~~Strikethrough~~ for deletions

## Colors

- {red}Red text for warnings{/red}
- {blue}Blue text for information{/blue}
- {green}Green text for success{/green}

## Highlighting

- ==Yellow highlight for important notes==
- [[Cyan highlight for definitions]]
- {{Green highlight for completed items}}
- ((Orange highlight for pending items))
